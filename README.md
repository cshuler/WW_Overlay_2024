# Hawaiʻi Cesspool Prioritization Tool — Act 217 Wastewater Overlay

![Act 217 Sewer Expansion Overlay Tool](https://raw.githubusercontent.com/cshuler/WW_Overlay_2024/main/Scripts/Figures/Cover_Shot.jpg)

## Overview
This repository contains the data, code, and documentation supporting the Act 217 Sewer Expansion Overlay of the Hawaiʻi Cesspool Prioritization Tool (HCPT), developed by the University of Hawaiʻi Water Resources Research Center and UH Sea Grant College Program. The overlay was mandated by Hawaiʻi Act 217 (HB2743, 2024), which directs the state to identify where centralized sewer expansion is most technically and economically feasible as a cesspool replacement strategy ahead of the 2050 conversion deadline.

The interactive web tool is publicly accessible at **[https://seagrant.soest.hawaii.edu/cesspool-tool-wastewater-overlay/](https://seagrant.soest.hawaii.edu/cesspool-tool-wastewater-overlay/)**.

More information about the Hawaii Cesspool Prioritization TOol is availabe at **[hawaiicesspooltool.org](http://www.hawaiicesspooltool.org)**.

## Repository Contents
- **Data/** — Master project areas spreadsheet, cost model data, and GIS-derived inputs
- **Scripts/** — Python scripts and Jupyter notebooks used for figure generation and analysis
- **Figures/** — All figures included in the legislative report

## The Tool
The Act 217 Sewer Expansion Overlay identifies 103 Potential Project Areas statewide covering approximately 39,300 cesspools. Each area is assigned a planning status and a cost feasibility tier based on a planning-level cost model with five infrastructure components: collection pipe, conveyance, treatment, neighborhood pump stations, and regional pump stations.
